3. ОБНОВИТЬ ResetPassword.tsx
==========================================

Файл:

artifacts/palata/src/pages/ResetPassword.tsx

Полностью заменить текущую заглушку.

Страница должна читать из URL:

token
email

Пример:

/reset-password?token=...&email=user%40example.com

Форма:

- заголовок «Новый пароль»;
- отображение email;
- поле «Новый пароль»;
- поле «Повторите новый пароль»;
- кнопки показать/скрыть пароль;
- кнопка «Сохранить новый пароль»;
- индикатор выполнения;
- понятные ошибки.

Клиентская проверка:

- token обязателен;
- пароль минимум 8 символов;
- минимум одна буква;
- минимум одна цифра;
- оба пароля должны совпадать.

При отправке вызвать:

resetPassword(token, password, passwordConfirmation)

После успешного ответа:

перейти на

/login?passwordReset=1&email=<email из ответа API>

Использовать useLocation из wouter.

Не выполнять автоматический вход.

Не сохранять пароль.

Не передавать пароль в URL.
