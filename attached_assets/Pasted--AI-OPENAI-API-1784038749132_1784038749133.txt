6. Безопасность
==================================================

Подтвердить:

- OPENAI_API_KEY используется только на backend;
- ключ не попадает в VITE_*;
- ключ не возвращается frontend;
- ключ не логируется;
- frontend не вызывает OpenAI напрямую;
- файл базы знаний не содержит секретов;
- endpoint имеет необходимую авторизацию или ограничения.

Если endpoint доступен анонимно — только отметить это как риск, код пока не менять.
